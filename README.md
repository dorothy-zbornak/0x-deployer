#0x-deployer

Create a `secrets.json` file at the root of this project:
```json
{
    "deployerKey": "DEPLOYER_ACCOUNT_PRIVATE_KEY",
    "etherscanKey": "YOUR_ETHERSCAN_API_KEY"
}
```

You will need to modify `src/deploy.js` to suit whatever you're trying to deploy.

Example usage:

```bash
cd /0x-monorepo/contracts/asset-proxy
yarn build && yarn link
cd /0x-deployer
yarn add '@0x/contracts-asset-proxy'
yarn deploy
```
